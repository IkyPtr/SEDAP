Perkenalan Tailwind CSS

Tailwind CSS merupakan framework utility-first yang memungkinkan kita membangun UI dengan cepat menggunakan kelas CSS yang sudah disediakan, tanpa perlu membuat class CSS manual.

CSS Manual

.div {
	background-color:red;
	padding-right:5px;
}

Tailwind CSS

<div class="bg-red-50 pr-4">

Class yang sudah disediakan oleh tailwindcss sudah sangat lengkap sehingga kita bisa membuat styling yang sederhana maupun styling mengikuti desain yang kita miliki.

Kategori Tailwind CSS

Tailwind CSS menawarkan fleksibilitas dalam kustomisasi desain. Agar lebih mudah dipahami dan digunakan secara efektif, kita akan membaginya ke dalam beberapa kategori sebagai berikut:

1️⃣ Spacing (Margin & Padding)

🔹 Mengatur jarak luar (margin) dan jarak dalam (padding) elemen.

  • m-4 → Margin 1rem
  • p-2 → Padding 0.5re
  • mx-4 → Margin horizontal 1rem (left & right)
  • py-3 → Padding vertical 0.75rem (top & bottom)

Contoh: 

function Spacing(){
    return (
        <div className="bg-white shadow-lg p-6 m-4 rounded-lg">
            <h2 className="text-lg font-semibold">Card Title</h2>
            <p className="mt-2 text-gray-600">Ini adalah contoh penggunaan padding dan margin di Tailwind.</p>
        </div>
    )
}

2️⃣ Typography (Teks & Font)

🔹 Mengatur ukuran, ketebalan, warna, dan gaya teks.

  • text-lg → Ukuran teks besar
  • font-bold → Teks tebal
  • text-gray-500 → Warna abu-abu
  • uppercase → Huruf kapital semua

Contoh:

function Typography(){
    return (
        <div>
            <h1 className="text-3xl font-bold text-blue-600">Tailwind Typography</h1>
            <p className="text-gray-600 text-lg mt-2">Belajar Tailwind sangat menyenangkan dan cepat!</p>
        </div>
    )
}

3️⃣ Border & Radius (Bentuk Elemen)

🔹 Menambahkan garis tepi dan membuat sudut membulat.

  • border → Garis tepi default
  • border-2 → Tebal border 2px
  • border-red-500 → Warna border merah
  • rounded-lg → Membuat sudut membulat

Contoh:

function BorderRadius(){
    return (
        <button className="border-2 border-blue-500 text-blue-500 px-4 py-2 rounded-lg"> Klik Saya </button>
    )
}

4️⃣ Background & Colors

🔹 Mengatur warna latar belakang dan teks.

  • bg-blue-500 → Latar belakang biru
  • bg-gradient-to-r from-green-400 to-blue-500 → Gradien hijau ke biru
  • text-white → Teks putih

Contoh:

function BackgroundColors(){
    return(
        <div className="bg-blue-500 text-white p-6 rounded-lg shadow-lg">
            <h3 className="text-xl font-bold">Tailwind Colors</h3>
            <p className="mt-2">Belajar Tailwind itu seru dan fleksibel!</p>
        </div>
    )
}

5️⃣ Flexbox & Grid (Layouting)

🔹 Membantu menyusun tata letak dengan mudah.

  • flex → Mengaktifkan Flexbox
  • justify-between → Menyebarkan elemen ke ujung
  • items-center → Pusatkan elemen secara vertikal
  • grid grid-cols-3 → Membuat layout 3 kolom

Contoh: 

function FlexboxGrid(){
    return (
        <nav className="flex justify-between bg-gray-800 p-4 text-white">
            <h1 className="text-lg font-bold">MyWebsite</h1>
            <ul className="flex space-x-4">
                <li><a href="#">Home</a></li>
                <li><a href="#">About</a></li>
                <li><a href="#">Contact</a></li>
            </ul>
        </nav>
    )
}

6️⃣ Shadow & Effects

🔹 Menambahkan bayangan dan efek visual lain.

  • shadow-md → Bayangan sedang
  • shadow-lg → Bayangan besar
  • hover:shadow-xl → Efek bayangan saat hover
  • opacity-50 → Transparansi 50%

Contoh: 

function ShadowEffects(){
    return (
        <div className="bg-white shadow-lg p-6 rounded-lg hover:shadow-2xl transition">
            <h3 className="text-xl font-semibold">Hover me!</h3>
            <p className="text-gray-600 mt-2">Lihat efek bayangan saat hover.</p>
        </div>
    )
}

Reusable Component

Reusable Component merupakan suatu komponen yang bisa digunakan berulang kali, sehingga pengembangan bisa lebih efisien dan terstruktur.

Silahkan buat sebuah file baru dengan nama UserForm.jsx lalu masukkan kode berikut

React Form dengan State

Selanjutnya kita akan menangani input dari user melalu sebuah form lalu menyimpan inputan tersebut ke State melalui Event Handling dan menampilkan UI sesuai dengan value State.


Buat sebuah file baru dengan nama
HitungGajiForm.jsx lalu isi dengan kode berikut:

  • HitungGajiForm.jsx
    export default function HitungGajiForm() {
	return (
		<div className="flex flex-col items-center justify-center m-5 p-5 bg-gray-100">
			<div className="bg-white p-6 rounded-lg shadow-lg w-96">
				<h2 className="text-2xl font-semibold text-center mb-4 text-gray-700">Hitung Gaji Bersih</h2>

				<div className="mb-4">
					<label className="block text-gray-700 font-medium mb-1">
						Gaji Pokok
					</label>
					<input
						type="number"
						placeholder="Masukkan jumlah gaji"
						className="w-full p-2 border border-gray-300 rounded focus:ring-2 focus:ring-blue-500"
					/>
				</div>
				<div className="mb-4">
					<label className="block text-gray-700 font-medium mb-1">
						Pajak: <b class="text-red-500">11%</b>
					</label>
				</div>

				<div className="mt-4 p-3 bg-blue-100 border-l-4 border-blue-500 text-blue-700">
					<p className="font-semibold">
						Total Take Home Pay (THP): Rp 0
					</p>
				</div>
			</div>
		</div>
	);
}

Kita akan menerapkan State untuk melakukan perhitungan THP berdasarkan ketikkan User dan nominal pajak yang ditetapkan. Sesuaikan kode diatas dengan menerapkan perubahan berikut:

  1. Deklarasikan State gaji dan pengolahan perhitungan:
    • gaji merupakan variabel yang menyimpan nilai inputan pengguna.
    • setGaji adalah fungsi yang dipanggil saat event onChange digunakan untuk memperbarui value dari variabelgaji.
    • useState("") berarti nilai awal dari variabel gajiadalah string kosong ("").
    • pajak dan totalGaji merupakan rule dalam pengolahan data gaji

  1. Sesuaikan inputan Gaji dengan menerapkan Event Handling onChange :
    • Jenis Event Handling-nya berupa onChange pada inputan gaji,
    • Setiap user melakukan input, function setGaji() akan dipanggil dengan menjadikan value ketikkan sebagai parameter-nya
    • e.target.value ini digunakan mendapatkan value ketikkan user
    • State gaji akan diperbaharui secara otomatis setiap kali user mengetik di inputan tersebut

  1. Sesuaikan bagian THP dengan mengambil dari variabel totalGaji
    • Pada bagian ini totalGaji ditampilkan dengan menerapkan toLocaleString() agar angka yang tampil lengkap dengan pemisah ribuan

  1. Proses Re-render ulang UI akan terjadi setiap perubahan State

    Ketika State gaji berubah saat adanya ketikkan oleh User, React akan me re-render ulang komponen HitungGajiForm() sehingga perhitungan totalGaji yang juga ada di function tersebut akan dihitung kembali dan di-update ke UI yang menampilkannya.

Conditional Rendering

Sebelumnya kita sudah mengetahui bahwa pada komponen HitungGajiForm.jsx memiliki sebuah state gaji yang menyimpan nominal dari gaji yang diinputkan.

Sekarang kita akan coba pasang kondisional sebagai berikut:

  • Jika gaji belum diinputkan atau bernilai 0, maka tampilkan informasi “Silakan masukkan gaji yang valid (tidak boleh kosong atau negatif)”
  • Jika gaji berisi nilai yang valid, maka tampilkan informasi hasil perhitungan THP
